В рамках узучения git я восстановил аккаунт на гитхабе, со времен динозавров аккаунт
Создал новый репозиторий, создал в нем изначально файл README, где указал "Здесь будет все что связано с ИПР и заданиями"
Через команду  git clone https://github.com/IannDior/IPR.git - сделал клонирование папки из удаленного репозитория в локальный 
Перед следующим шагом добавим описание в README, чтобы проверить что наш коммит и пуш работают
Длаее через команду git add . - обновиляем полностью дирректорию файлами, которые будем коммитить в удаленный репозиторий
Делаем далее свой первый commit через git commit -m "first" -m здесь означает message, мы делаем коммит с сообщением, коммитом мы фиксируем изменения которые будут отправлены через push
И делая git push мы отправляем все внесенные изменения на наш удаленный репозиторий
