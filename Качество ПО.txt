Качество ПО — это степень соответствия программного продукта ожиданиям пользователей, требованиям заказчика и стандартам. Оно оценивается по различным критериям, которые помогают определить, насколько продукт надежен, удобен, функционален и соответствует поставленным целям

Основнеые критерии качества ПО:
1. Функциональность - способность ПО выполнять заявленные функции 
Сюда входит: полнота (все реализовано), корректность (правильность выполнения), совместимость (работа с другими системами)
2.Надежность - способность выполнять функции в заданных условиях 
Сюда входит: Стабильность, восстанавливаемость, отказоустойчивость
3. Удобство использования - легкость при взаимодействии с ПО
Сюда входит: понятность интерфейса, простота в использовании
4. Производительность - скорость и эффективность работы 
Сюда входит: время отклика, масштабируемость
5. Безопасность - защита данных системы
Сюда входит: аутентификация, шифрование данных, устойчивость к атакам

Методы обеспечения качества ПО:
1. Тестирование - UI/UX, функциональное, регрессионное и т.д
2. Автоматизация тестирования
3. Документация - четко описанные требования и процессы
4. Code review - проверка кода на ошибки, сюда же можно добавить unit test - для более точной проверки кода
